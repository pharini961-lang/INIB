## 🚀 Project 1: Signup & Login Pages (Light & Dark Mode)
**Objective:** Design a seamless authentication flow utilizing distinct visual themes for light and dark modes.

### Key Features:
* **Dynamic Backgrounds:** Leveraged carefully selected background imagery to emphasize the contrast and mood shifting between themes.
* **Accessibility:** Ensured high text-to-background contrast ratios for readability in both modes.

I have not just replicated any login or sign up page. I used images as a background to imply the theme of my work (Boutique's login and signup page). I used a medium-dark image so that the viewer does not strain their eyes, and for Light mode I used a creamy white image as a background image which looks bright and fits the theme.

## 🛒 Project 2: Interactive E-Commerce Product Page
**Objective:** Create an engaging, conversion-focused product description page with smooth interactions.

### Features Implemented:
* **Smooth Image Carousel:** Fully functional carousel allowing users to tap through product images with seamless *Dissolve* transitions.
* **Dynamic Pagination Dots:** Interactive active-state tracking indicator dots that change color to reflect the currently viewed image.
* **Horizontal Size Selector:** Integrated a clear, interactive size row (S, M, L, XL, XXL) featuring a distinct "selected state" design.
* **Success State Overlay:** A clean, custom success notification pop-up triggered seamlessly upon tapping the "ADD TO CART" button.

## 🍕 Project 3 (Major): 13-Screen Food Delivery App
**Objective:** Design an end-to-end food delivery mobile application covering the entire user journey from onboarding to order fulfillment.

### The User Journey (13 Screens):
1. **Splash & Onboarding:** Captivating entry screens to welcome the user.
2. **Authentication:** Quick login/signup.
3. **Home & Discovery:** Menu browsing, categories (Burgers, Pizza), and smart search.
4. **Product Detail:** Customizing orders and viewing ingredients.
5. **Cart & Checkout:** Order review, address selection, and payment options.
6. **Live Order Tracking:** Real-time updates from the kitchen to the doorstep.
